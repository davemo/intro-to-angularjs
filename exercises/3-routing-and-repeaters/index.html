Routing & Repeaters

  1. Add `ngRoute` as a dependency to your `app` module and inject $routeProvider during the `config` phase.
  2. Add an `ng-view` directive to div#view
  3. Configure `$routeProvider` with a route to /stooges that loads a template, stooges.html and assigns the controller FirstController
  4. Add an array of the 3 stooges (larry, curly, and moe) to the `$scope` for `FirstController`
  5. Use `ng-repeat` on a table#stooges element in stooges.html to display the list of stooges

Note: Open your browsers developer tools console and make sure the tests are passing.
